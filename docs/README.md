# 文档导航

按任务场景快速定位需要的说明。

- **部署起步**：`deployment-checklist.md`（从裸机到跑通样例的逐步检查）
- **硬件差异**：`hardware-notes.md`（记录驱动/固件/算子差异）
- **故障排查**：`troubleshooting.md`（现象-诊断-解决表）
- **Profiling**：`profiling.md`（CANN / MT 工具使用占位）
- **监控告警**：`monitoring.md`（采集指标与阈值参考）
- **离线/内网**：`offline-install.md`（air-gap 交付流程）
- **数据与权重**：`data-management.md`（路径约定、权限、清理）
- **参考资料**：`related-articles.md`（外部文章与摘要占位）

> 使用建议：按 “部署起步 → 硬件差异 → 自检脚本 → 示例运行 → 基线记录 → 监控/Profiling” 的顺序完成首轮验证。***
